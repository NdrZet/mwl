# Welcome to Omni Project

Omni Project — это современный гибридный (офлайн/онлайн) музыкальный стриминг-плеер, построенный на основе React, TypeScript, Electron и Node.js. Приложение предоставляет полнофункциональный интерфейс для управления музыкальной библиотекой, позволяет загружать треки локально, а также стримить музыку с собственного бэкенда.

## 🚀 Ключевые возможности:

- **Гибридное воспроизведение**: Стриминг аудиофайлов (MP3, WAV, OGG, M4A, FLAC, AAC) с облачного сервера (Node.js) + поддержка локального воспроизведения с жесткого диска.
- **Облачная синхронизация**: Авторизация и хранение библиотеки (лайки, плейлисты) через Firebase Firestore.
- **Умная категоризация**: Автоматическое извлечение метаданных (название, исполнитель, обложки) и группировка по Альбомам и Артистам.
- **Глобальный поиск**: Мощный поиск по всем трекам, альбомам и исполнителям.
- **Интеграция текстов**: Поддержка текстов песен (через сторонние API) с возможностью перевода на лету.
- **Радио и Подкасты**: Встроенная поддержка интернет-радиостанций и подкастов.
- **Современный UI**: Адаптивный и красивый интерфейс на базе Tailwind CSS и shadcn/ui.

## 🔧 Технологический стек

- **Frontend:** React 18 + TypeScript + Vite 5
- **UI Framework:** Tailwind CSS + shadcn/ui
- **Desktop Wrapper:** Electron 30
- **Backend Server:** Node.js + Express (стриминг музыки и проксирование переводов)
- **Database & Auth:** Firebase (Authentication, Firestore)
- **Audio Processing:** music-metadata (для извлечения метаданных)
- **State Management:** React Context API

## 📂 Структура проекта
```text
Omni_Project/
├── src/
│   ├── components/           # React компоненты
│   │   ├── ui/               # UI компоненты (shadcn/ui)
│   │   ├── MusicContext.tsx  # Главный контекст и логика объединения локальной/облачной базы
│   │   ├── MusicPlayer.tsx   # Плеер с поддержкой текстов песен
│   │   ├── GlobalSearch.tsx  # Глобальный поиск
│   │   └── ...               # Остальные компоненты (ArtistsGrid, AlbumsGrid и т.д.)
│   ├── styles/
│   │   └── globals.css       # Глобальные стили (Цветовая палитра Vision Lumina)
│   ├── firebase.ts           # Конфигурация подключения к БД
│   └── App.tsx               # Главный компонент и роутинг
├── server/
│   └── index.js              # Node.js бэкенд (стриминг музыки по кускам, API переводов)
├── electron.cjs              # Electron main process (доступ к ФС, интеграция локальных треков)
├── preload.js                # Electron preload script (безопасный мост IPC)
└── package.json              # Зависимости и скрипты
```
